About This Page

This website has been created by Michael Woods as a means of practicing the fundamentals of front-end development. This page has been created using basic HTML and CSS practices. With the help of the Bootstrap 5 framework, the page has developed into a user-friendly website that is also visually appealing and easily replicable in other contexts.

Bootstrap 5

Bootstrap is the industry leader in HTML, CSS, and Javascript frameworks. It is completely free to use and comes with several intuitive components that offer its user a wide range of freedoms in how they want to create their websites. This, along with its new edition, Bootstrap 5, has allowed it to be a great tool for both beginners and advanced developers.

Bootstrap is completely free to use and offers many tutorials on how to use the framework. Simply click the image or go to their homepage to try Bootstrap yourself.

Creating with BootStrap 5

With Bootstrap being one of the most popular free, open-source frameworks for web development, there is a large quantity of tutorials and documentation offered online. This makes developing in Bootstrap highly preferable for beginners. Bootstrap even offers multiple templates for using their wide array of components that can help make your websites look professional.

One of the best methods for learning more about Bootstrap is to watch the many tutorials offered on Youtube. freeCodeCamp.org and Traversy Media are two creators on Youtube that offer extensive tutorials on using BootStrap 5. This video here by freeCodeCamp.org has been incredibly helpful in both the creation and design of this page.
